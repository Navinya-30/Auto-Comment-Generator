# Changelog

All notable changes to the **Auto Comment Generator** extension.

## [0.1.0] - 2024

### Added
- Initial release
- Rule-based comment generation for JavaScript and Python
- Detects: functions, classes, loops, conditionals, try/catch, imports, variables
- Multi-language comment style support (`//`, `#`, `<!-- -->`, `/* */`, `--`)
- Command Palette integration: `Generate Comment`
- Keyboard shortcut: `Ctrl+Alt+C` (Windows/Linux), `Cmd+Alt+C` (Mac)
- Right-click context menu entry
- Friendly error messages when no code is selected
